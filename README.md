# Le Pendu

Le Pendu est un jeu consistant à trouver un mot en devinant quelles sont les lettres qui le composent.

### Features

- Deviner un mot
- Comptez le nombre d’essais et affichez-le au fur et à mesure.
- Interdire un nouvel essai d’une lettre,

### Technologies used:

- React Js
- Bootstrap
